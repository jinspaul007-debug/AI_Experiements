# 📱 Mobile-Only Deployment Guide

## Quick 5-Minute Setup from Office Laptop

### Option 1: GitHub Pages (Recommended)
1. **Open Browser**: Go to https://github.com (no software needed)
2. **Sign Up**: Create free account (2 minutes)
3. **New Repository**: Click "+" → "New repository"
   - Name: `lifestyle-tracker`
   - Public: ✅ (required for free hosting)
4. **Upload Files**:
   - Click "uploading an existing file"
   - Drag ALL Lifestyle folder files into browser
   - Click "Commit changes"
5. **Enable Pages**:
   - Go to Settings → Pages
   - Source: "Deploy from a branch" → Save
6. **Get URL**: After 1-2 minutes, you'll see URL like:
   `https://yourusername.github.io/lifestyle-tracker`

### Option 2: Netlify Drop (Instant)
1. Go to https://app.netlify.com/drop
2. Drag ALL Lifestyle files onto the page
3. Get instant URL (no account needed)

### Option 3: Vercel
1. Go to https://vercel.com
2. Sign up free
3. Import project → Drag files → Deploy

## iPhone Setup (2 minutes)

1. **Open URL** in Safari (from any of the options above)
2. **Share** → **Add to Home Screen**
3. **Name**: "LifeTrack" → **Add**
4. **Icon** appears on home screen like native app
5. **Works offline** after first load

## Backup Strategy (Mobile Only)

### Automatic Weekly Backup
1. Settings → Export JSON
2. Save to Files → Google Drive
3. Set weekly reminder in iPhone Calendar

### Monthly Backup
1. Export CSV to Files
2. Email to yourself
3. Save to iCloud Drive

## Troubleshooting

### App Not Loading?
- Clear Safari cache: Settings → Safari → Clear History
- Re-add to home screen
- Check internet connection (first load only)

### Data Lost?
- Import from latest JSON backup
- Check iPhone storage (Settings → iPhone Storage)

### Need to Update App?
- Upload new files to GitHub
- Wait 2 minutes for deployment
- Refresh iPhone app (pull down)

## Office Laptop Tips

### If GitHub Blocked
- Use personal email for GitHub
- Try Netlify Drop (no account needed)
- Use mobile hotspot from iPhone to upload

### If USB Disabled
- Upload via email to personal account
- Use cloud storage (Google Drive, OneDrive)
- Transfer via AirDrop to personal device

### If Browser Restricted
- Try different browser (Chrome, Edge, Firefox)
- Use mobile hotspot to bypass restrictions
- Upload from home network
