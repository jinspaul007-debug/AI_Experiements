# 📖 User Guide — Lifestyle Challenge Tracker v3.0

A step-by-step guide to using every feature. The app is 100% local (your data never leaves your device) and works offline once loaded.

---

## 1. First Launch — Create Your Profile
1. On first open you land on the **Profile** page.
2. Pick an **avatar**, type your **name**, tap **🚀 Create Profile**.
3. You're taken to the **Challenge** page to start your first challenge.

> Multiple people can use the same device — each gets a separate profile with isolated data.

---

## 2. Start a Challenge
1. On the **Challenge** page, pick a **type**:
   - 🌟 Lifestyle, ⚖️ Weight Loss, 😴 Sleep, 🏃 Activity, 🍎 Calorie, ✏️ Custom
2. (Optional) Give it a **name**.
3. Choose a **Duration Mode**:
   - **Fixed number of days** — type any number (e.g. 30, 90, 101).
   - **Start date → End date** — pick a date range.
4. (Optional) Set a **Goal Weight**.
5. For **Custom** challenges: tick the habits you want, and add your own with **+ Add**.
6. Tap **🚀 Start Challenge**.

---

## 3. Track Your Day
1. Tap **Track** in the bottom bar.
2. Use the **‹ ›** arrows to move between days (within the challenge range).
3. Tick habits, fill in numbers (water, sleep, steps, weight…), pick a **day type**, set **mood/energy**, add **notes**.
4. For calorie/weight challenges, log **meals** (name + kcal + type) — daily total vs target shows automatically.
5. Tap **💾 Save Day**. To clear a day, tap **🗑️**.

---

## 4. Health Tools (More → 🧮 Health Tools)
1. **Body Metrics** — enter height, age, gender, activity level, start & goal weight, then **Save**. These power all calculators.
2. **BMI Calculator** — get your BMI, category, and healthy weight range.
3. **Maintenance Calories (TDEE)** — how many calories you burn per day.
4. **Calorie Burn** — pick an activity + minutes to estimate calories burned.
5. **Weight-Loss / Gain Planner** — enter current weight, goal weight, and days. You get:
   - daily calorie target, daily deficit/surplus, weekly pace, a projection chart, and safety warnings.
   - Tap **🎯 Save as Goal & Challenge** to track it.

---

## 5. Goals (bottom bar → Goals)
1. Tap **➕ Add a Goal**, choose a type (water, sleep, reading, study, steps, weight).
2. Set the **target**, **start** and **end** dates (weight goals also ask current weight).
3. Each goal shows:
   - a **progress bar**, a **pace badge** (🚀 Ahead / ✅ On Track / ⚠️ Behind),
   - a **projection chart** (dashed = planned, solid = your actual data), and a projected finish.
4. Your top goals also appear in **Goals at a Glance** on the dashboard.

---

## 6. Analytics (Stats)
Trend charts (with target lines), habit heat-map, mood/energy, calorie & steps, and a weekly review with suggestions.

---

## 7. Compare With Others (More → 🔀 Compare)
1. **Export for Sharing** — creates a privacy-filtered file (only the data types you allow in Settings → Privacy).
2. Send it to a friend; they **Import** it on their device, and vice-versa.
3. See side-by-side average scores and a comparison chart.

---

## 8. Privacy & PIN (More → ⚙️ Settings)
- **Privacy & Sharing** — toggle exactly which data types are included when you export for sharing.
- **PIN Lock** — tap **Set a PIN** to require a 4-digit code on app open and profile switch. You can change or remove it later. *Forgot PIN* can only delete that profile (protects your privacy).

---

## 9. Backup & Restore (Settings → Data Management)
- **Export JSON / CSV** — current profile's data.
- **🗄️ Backup ALL Profiles** — single file with every profile + all data.
- **♻️ Restore Backup** — re-import a full backup (overwrites current data).
- **Reset This Profile's Data** — clears just the active profile.

> Tip: back up monthly to iCloud / Google Drive.

---

## 10. Print / PDF (More → 🖨️ Print Sheets)
Generate A4-ready weekly sheets or a full report — use your browser's Print → Save as PDF.

---

## 11. Install on iPhone / Android (PWA)
1. Open the app URL in **Safari** (iPhone) or **Chrome** (Android).
2. **Share → Add to Home Screen**.
3. Launches full-screen like a native app and works offline.

---

## Tips
- Works on Chrome, Edge, and iPhone 12 Safari.
- Switch **Dark/Light** mode in Settings.
- All data is stored in your browser only — clearing browser data erases it, so keep backups.
